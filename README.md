# Be Quiet

音量を固定するためのアプリ
for Android

## Why?

〇〇PayやらP◯yP◯yやら、決済音を出すことを強制するのにデバイス自体のメディア音量をグローバルで上げてくるのですよ。
で、音量ゼロは無理だと。
そんなバカげた話許せないですよね。
じゃあ彼らと同じ手段で音量をグローバルでゼロにすればいいのではないか、ということです。

## How?

音量変化時にフックを...というのはレスポンスの関係で断念しました。
なので常駐で常に音量をゼロにする脳筋構成です。

そして音量ゼロだけではなく、音量レベル指定値での固定もできます。
途中、本アプリからの調整もできます。

## 必要権限

- FOREGROUND_SERVICE
  - 常駐サービスのため
- POST_NOTIFICATIONS
  - 常駐通知、アプリ復帰用
